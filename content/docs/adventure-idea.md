# 冒險靈感
如果你需要臨場發揮，或只是想找點創作啟發，以下是一些能加以延伸的冒險靈感。\
它們大多可以適配單次聚會的長度，並使用了怪物圖鑑中記述的怪物。

  .adventure-list li {
    margin: .8em 0;
  }
</style>

<div class="adventure-list">

- 某個村莊的水源被 **吞噬意識體** 汙染，居民們逐漸扭曲病變。感染的源頭位於封閉礦坑的深處，或在怪物盤據的山地或樹林中。
- 某個大城市的外圍地帶爆發了 **吞噬意識體** 的初期感染徵兆，引發大規模的逃難與混亂。
- 某個宣稱能賜予永生的 **扭曲崇拜者** 穿梭各個城鎮之間，毒害那些遭受它魔爪的可憐人。
- 某座漁村在漁獲枯竭的艱難時期轉而膜拜 **夢眠海德拉**，將不願加入邪教的人們獻祭以換取豐收。
- 某片海域有一頭 **斯庫拉** 會襲擊經過其地盤的船隻 ── 有些人因此開始主動以活祭換取通行平安。
- 調查顯示一個 **深淵子嗣** 沉睡在某個聚落的附近，且正被一群崇拜祂的密教徒逐漸喚醒。
- 為了施放魔咒，需要採集某種罕見的稀有植物，但其生長的樹林盤據著 **阿剌克涅**、**獅鷲**、**蠍尾獅** 和其他危險的野獸。
- 一頭 **龍** 因為某個原因被喚醒，並威脅著周遭地區 ── 即使順利安撫牠，使其回歸沉睡，將牠喚醒的源頭仍然必須被處置。
- 某座古老的修道院保存著重要的魂縛聖物，但如今卻被一隻 **蛇髮女妖** 給佔據為巢。
- 一頭 **狼人領主** 試圖透過感染他者，擴大自己的狼群規模。
- 某個商隊遭到強盜襲擊 ── 但真相是，其中一位商隊護衛其實是 **狼人**。
- **石像鬼**、**尖嘯魔**，和其他惡魔出現在某座城鎮的夜晚，擄走街道上的行人，帶到未知的地點。
- 某間學院或修道院被惡魔們佔領，院內眾人受困其中 ── 其中肯定也包括召喚者本人。
- 惡魔大公 **瑪法斯** 潛伏在某座城鎮中，蒐集人們的秘密用來勒索抹黑，煽動居民們彼此敵對，企圖讓整座城鎮分崩離析。
- 一頭 **屠戮惡魔** 在某座城市的地下隧道築巢，獵殺那些誤闖其中的可憐人。
- 一名 **魅魔** 操弄著地區領袖，以貪婪和猜忌之名施行高壓律法。
- 一株 **愛娜溫花妖** 秘密扎根在某位貴族的大花園或城市角落中，悄悄襲擊那些獨自靠近她的人。
- 一個 **二重身** 偽裝成其中一位英雄或其親友，試圖破壞他的人際關係或用其身分犯罪。
- 自從妖精領主對森林邊境的某座村莊發出警告，要求其立即搬離之後，一名 **杜拉罕** 每晚都會來到村莊殺死一個人，直到居民們屈服於牠的要求。
- 一齣高調宣傳的戲劇表演即將上演，其創作內容清楚帶有 **黃金繆思** 留下的痕跡。
- 一群 **紅帽子** 藏身在城市內的某座廢棄建築內，嗜血行徑越發肆無忌憚。
- 自然荒野的邊界每天都在逐步靠近村莊，樹海不斷增長，並由 **樹人** 們擔任最前線。
- 在某棟兇宅中作祟騷擾房客的 **幽靈** 生前是被另一隻怪物殺害，至今仍逍遙法外。
- 一位知名的醫生被發現慘遭謀殺 ── 而他親手創造的 **人造魔骸** 也下落不明。
- 瘟疫開始在城市中蔓延。這是普通的傳染病、**諾思費拉圖** 的陰謀、還是其他怪物的傑作？
- 一名 **吸血鬼領主** 發出號召，將來自四海的怪物集結於邪惡的旗幟之下。
