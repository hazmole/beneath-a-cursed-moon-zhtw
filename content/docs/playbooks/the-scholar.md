# 學者 Scholar

{{< playbook-desc >}}
知識是對抗黑暗最強大的武器之一，而有些人也將其視為利劍揮舞。\
無論是受過教育的學生、精明的商人，甚至是前邪教徒，\
都能將苦心鑽研的成果轉化為力量，用以戰勝自己肉體無法對抗的恐懼。

他們的知識是無價的資源——他們隨身攜帶的種種工具裝備，以及他們涉獵的魔法也是如此。
{{< /playbook-desc >}}

  - 當他們在你附近調查時，而你正好擁有他們所需的事物。
  - 當你協助指導其中某個人時。
  - 當你們被同一個組織指派共同行動時。

## 基本能力
### 〈百寶袋〉{.move-title}
{{< move-block >}}
你擁有一個裝滿道具的袋子，能用於解決各種情境。每場冒險五次，你可以拿出以下列表中的任何道具。其中任何以複數形式呈現的道具都足以供單人隨時取用，並維持長期消耗：
- 一把銀製匕首（<etag>3 傷害</etag> <etag>近身</etag> <etag>隱蔽</etag> <etag>銀</etag>）
- 一把鐵製匕首（<etag>3 傷害</etag> <etag>近身</etag> <etag>隱蔽</etag> <etag>鐵</etag>）
- 數支銀製或鐵製彈藥
- 一把重型火炬（<etag>3 傷害</etag> <etag>近身</etag> <etag>火焰</etag>）
- 一把家傳寶劍（<etag>3 傷害</etag> <etag>近身</etag> <etag>魂縛</etag>）
- 數瓶燃燒瓶（<etag>2 傷害</etag> <etag>遠程</etag> <etag>火焰</etag>，目標在撲滅火焰前每次行動都會受到無視護甲的 1 傷害）
- 一件火焰聖徽（驅魔工具）
- 一件鐵製聖徽（驅魔工具）
- 一件銀製聖徽（驅魔工具）
- 一件魂縛聖徽（驅魔工具）
- 一小瓶純淨靈魂精華（讓一把武器在一個場景期間得到 <etag>魂縛</etag> 標籤）
- 一小瓶燃油（讓一把武器在一個場景期間得到 <etag>火焰</etag> 標籤）
- 一卷記載了任意儀式或戰鬥巫術的卷軸
{{</ move-block >}}

### 〈致命好奇心〉{.move-title}
{{< move-block >}} 
每當你因為想要獨自調查或掌握新知，而在沒有其他團隊成員支援的情況下使自己陷入危險，你獲得 2 掌握，持續直到整個冒險結束。你可以按照〈理解戰況〉、〈通曉傳說〉或〈四處探查〉動作的規則來消耗這些掌握。
{{</ move-block >}}



## 進階能力
### 〈萬全準備〉{.move-title .withCheckbox}
{{< move-block >}} 
你的〈百寶袋〉在每場冒險中可以額外使用 3 次。此外，你可以消耗 2 次〈百寶袋〉的使用次數，來拿出任何你能夠合理隨身攜帶的道具。
{{</ move-block >}} 

### 〈初窺星界奧秘〉{.move-title .withCheckbox}
{{< move-block >}} 
你在〈星界巫術〉的加值增加到 +2。\
你知曉二種該學派魔法的戰鬥魔咒；且每場冒險一次，你可以施展一次儀式巫術魔咒而無須借助卷軸或其他外部資源的協助。
{{</ move-block >}} 

### 〈初窺煉獄奧秘〉{.move-title .withCheckbox}
{{< move-block >}} 
你在〈煉獄巫術〉的加值增加到 +2。\
你知曉二種該學派魔法的戰鬥魔咒；且每場冒險一次，你可以施展一次儀式巫術魔咒而無須借助卷軸或其他外部資源的協助。
{{</ move-block >}} 

### 〈初窺死靈奧秘〉{.move-title .withCheckbox}
{{< move-block >}} 
你在〈死靈巫術〉的加值增加到 +2。\
你知曉二種該學派魔法的戰鬥魔咒；且每場冒險一次，你可以施展一次儀式巫術魔咒而無須借助卷軸或其他外部資源的協助。
{{</ move-block >}} 

### 〈笨手笨腳〉{.move-title .withCheckbox}
{{< move-block >}} 
每當你要承受〈大膽行動〉的負面後果時，你總是可以選擇失去你〈百寶袋〉的一次使用次數，或讓你已經拿出的一件道具遺失或損壞來當成你要承受的後果。你也能透過這個方式使攻擊你造成的傷害減輕 2 點。
{{</ move-block >}} 

### 〈戲劇性揭露〉{.move-title .withCheckbox}
{{< move-block >}} 
每當你從〈百寶袋〉中拿出一件物品並立即使用，你在任何使用了該物品的動作中得到 單次+2。
{{</ move-block >}} 

### 〈輕裝甲〉{.move-title .withCheckbox}
{{< move-block >}} 
你投資了一套輕型皮甲或鎖子甲，能在穿戴時提供 1 護甲。
{{</ move-block >}} 

### 〈只不過是個學者！〉{.move-title .withCheckbox}
{{< move-block >}} 
你知道該如何讓自己看起來人畜無害。你可以消耗〈理解戰況〉的 1 掌握 來讓一個敵人的注意力從你身上轉到別處 ── 但你不能決定他的注意力會轉到哪裡去。
{{</ move-block >}} 

### 〈綽綽有餘〉{.move-title .withCheckbox}
{{< move-block >}} 
每當你從〈百寶袋〉中拿出道具時，你可以額外消耗一次使用次數，來拿出足以供整個隊伍使用的份量。
{{</ move-block >}}
