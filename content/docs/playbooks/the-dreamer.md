# 幻夢者 Dreamer

{{< playbook-desc >}}
來自星辰之外，深水之下，以及大地深處的沉睡者們，正不斷向那些能聽見祂們聲音的不幸之人低語。祂們的呢喃潛入夢境，侵蝕思想，帶來不被渴望的洞見。

有些人能掌控這些不請自來的秘密，將其化做星界魔法，在受選者的身軀內掀起浪潮。\
儘管浪潮能被駕馭，卻仍能改變它所觸及的一切，緩慢地，使幻夢者的心靈隨著一次次沖刷而崩解。
{{< /playbook-desc >}}


## 快速開局背景
{{% columns ratio="4:3" %}}
- <div class="withCheckbox subContentBlock">

  人們因為你的古怪習慣而感到不安：
  - 雙眼從不眨動。
  - 發出嘶啞咕嚕的呼吸聲。
  - 在睡夢中吟誦咒語。
  - 流出的鮮血宛如漆黑油液。

  沉睡者與你緊密連結，是因為：
  - 你曾經過於接近祂們沉睡之地。
  - 你曾染指那些不應被觸碰的禁忌魔法。
  - 祂們被你的藝術創作給吸引。

  儘管你行徑怪異，其他英雄仍然願意與你共事，因為：
  - 他們在你變異之前就認識你。
  - 你深信他們能幫助你平息腦中的低語。
  - 你證明了自己擁有拯救他人的可靠能力。
  - 幾乎沒有人比你更瞭解那些怪物。

## 基本能力
### 〈通神〉{.move-title}
{{< move-block >}}
你可以潛入夢境世界，窺探該地區生物的心靈。這個動作視為調查動作，使用你的 **〈星界巫術〉** 加值，並從以下問題中如常向 MC 詢問：

- 這裡沉睡著什麼樣的邪惡？
- 這裡曾經發生過什麼重大事件？
- 這裡徘徊著什麼樣的情緒和恐懼？
- 周遭地區和建築物有著什麼樣的地理環境和地形？
{{< /move-block >}}

### 〈星界術者〉{.move-title}
{{< move-block class="withCheckbox" >}}
你知曉所有星界戰鬥巫術，且如果你想要施展星界儀式巫術，你也清楚知道該儀式的內容，而無須再去尋找其相關紀錄或卷軸。
{{< /move-block >}}

### 〈蒙受異界恩典〉{.move-title}
{{< move-block class="withCheckbox" >}}
從以下選擇一項突變。你每得到三個進階能力，就可以再選擇一項突變。
- 能在徹底的黑暗中清楚視物。
- 能在水下呼吸，並完美地游泳。
- 忽略所有毒素、不適、疾病和腐化的影響。
- 無須睡眠。
- 能夠食用任何有機物質，無論該物質是否已經腐爛、變質或產生毒素。
{{< /move-block >}}


## 進階能力
### 〈禁忌的知識〉{.move-title .withCheckbox}
{{< move-block >}} 
每當你進行〈理解戰況〉或〈通曉傳說〉動作時，你都可以選擇汲取禁忌的知識。當你這麼做時，得到額外的 1 掌握，或如果你骰出 13+ 的結果則改為額外的 2 掌握。當你使用這個能力時，你在〈抵抗黑魔法〉動作將承受 持續-1，持續直到你有時間好好休息。
{{</ move-block >}} 

### 〈引領信眾〉{.move-title .withCheckbox}
{{< move-block >}} 
當你消耗來自〈理解戰況〉的 掌握 以增加對怪物造成的傷害時，你和任何與你並肩作戰的凡人們都能造成 +1 傷害，且在處理該效果的持續時間時不會被視作「下一次攻擊」。此外，每當你消耗〈理解戰況〉的 掌握 來讓你的同伴得到〈抵抗黑魔法〉的加值時，所有與你並肩作戰的凡人們也都將免疫該效果。
{{</ move-block >}} 

### 〈縈繞不散的靈氣〉{.move-title .withCheckbox}
{{< move-block >}} 
當你使用〈通神〉時，你可以改為專注在一個物體而非地區，詢問以下問題：

- 此物的上一任擁有者是誰？
- 自上個滿月以來，來有誰經手過此物？
- 哪一任擁有者與此物的連結最為深厚？
- 此物在何時，因為什麼原因被轉手？
- 此物曾被用來做什麼？或用在什麼身上？
{{</ move-block >}} 

### 〈月之誘惑〉{.move-title .withCheckbox}
{{< move-block >}} 
你的魔力與月相緊密相連。在滿月之夜，你施展的所有魔咒都能夠忽略戰鬥巫術和儀式巫術的副作用 ── 但與你一同參與儀式的同伴們則不然。在新月之夜，你的所有〈巫術〉動作都會受到 持續-1。在其他夜晚，你在每個場景中能夠忽略第一個戰鬥巫術的副作用。
{{</ move-block >}} 

### 〈惑導夢境〉{.move-title .withCheckbox}
{{< move-block >}} 
你可以進入出神狀態，遁入某個你在腦海中有清晰形象之人的夢境中。使用你的〈星界巫術〉加值擲骰：

若骰出 6 或以下，目標會夢見你，並產生疑心。\
若骰出 7-12，你可以在目標腦中植入一個念頭，他會付諸實行，直到明顯面臨巨大風險為止。\
若骰出 13+，目標會以狂熱奉獻的態度去實行被植入的念頭。\
若骰出 9 或以下，你也會額外受到一個儀式巫術的副作用。
{{</ move-block >}} 

### 〈巫咒學家〉{.move-title .withCheckbox}
{{< move-block >}} 
選擇一種魔法：煉獄、死靈。你使用該種類魔法的〈巫術〉加值增加到 +2。\
你知曉二種該學派魔法的戰鬥魔咒；且每場冒險一次，你可以施展一次儀式巫術魔咒而無須借助卷軸或其他外部資源的協助。
{{</ move-block >}} 

### 〈扭曲血肉〉{.move-title .withCheckbox}
{{< move-block >}} 
你的皮膚底下翻騰著扭曲的增生血肉。你得到針對物理打擊的 1 護甲，但這個護甲會被帶有 火焰 標籤的武器，或其他來源的傷害給無視。每當有生物以吞食你一部分（例如咬噬、吸血、奪取靈魂等）的方式來對你造成傷害，該生物會受到 1 點無視護甲的傷害。
{{</ move-block >}} 

### 〈沉睡者的低語〉{.move-title .withCheckbox}
{{< move-block >}} 
每當你陷入沉睡，或當你進行〈治療休養〉動作時，你可以聆聽沉睡者的低語，選擇以下其中一項好處：

- 額外治療 1 點傷害
- 在〈抵抗黑魔法〉中得到 +1 加值，持續直到你施放魔咒
- 在你的下一個調查動作中得到 當次+1。
{{</ move-block >}} 

### 〈獻身狂熱〉{.move-title .withCheckbox}
{{< move-block >}} 
當你進行〈浴血奮戰〉或〈大膽行動〉動作時，你可以追加最多 單次+3。接著，你將在所有〈巫術〉動作中承受 持續-1，持續直到你使用〈巫術〉加值擲骰的次數等同於你前面追加的加值數。\
如果你正在承受源自此動作的持續罰值，則你不能再使用此能力。
{{</ move-block >}}
