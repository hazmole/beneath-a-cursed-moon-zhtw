# 邪術士 Warlock

{{< playbook-desc >}}
煉獄魔法是如此誘人的力量泉源，能夠滿足使用者所想要的一切 ── 只要他們渴望著毀滅的話。

邪術士出賣了自己的靈魂，換取強大的力量以焚毀他們的一切擔憂 ── 連同他們的靈魂一起。\
有些人認為這是保護摯愛之人的必要犧牲，並掙扎著不讓自己進一步墮落；\
有些人則與過去對抗，希望能尋得救贖。
{{< /playbook-desc >}}

## 基本能力
### 〈煉獄術者〉{.move-title}
{{< move-block >}}
你知曉所有煉獄戰鬥巫術，且如果你想要施展煉獄儀式巫術，你也清楚知道該儀式的內容，而無須再去尋找其相關紀錄或卷軸。
{{</ move-block >}}

### 〈獻祭〉{.move-title}
{{< move-block >}}
你可以透過獻祭給煉獄中的強大惡魔來換取魔法力量。隨著你持續獻祭，祂們將賜予你更多力量，但對你的索求也將與日俱增。獻祭是一種複雜的儀式，並需要花費調查行動的額度才能施展。進行獻祭是一種動作，並會根據你獻上的祭品決定其加值，如下所示：

- 殺死活著的動物 = +0 加值
- 取得並摧毀某人的財產 = +1 加值
- 取得並摧毀某人擁有的動物 = +2 加值
- 取得並摧毀對某人而言無可取代的珍貴事物 = +3 加值
- 殺死活著的人類 = +4 加值

若骰出 6 或以下，黑暗諸神對你的祭品感到不悅，且你在該場冒險期間都不能再進行獻祭。\
若骰出 7-9，得到 1 掌握。\
若骰出 10-12，得到 2 掌握。\
若骰出 13+，得到 3 掌握。

你可以每消耗 1 掌握來使你的〈巫術〉動作得到 單次+1，且在同一次擲骰中可以消耗任意數量的掌握。如果你試圖在同一場冒險中舉行多次獻祭，則你會承受減值，數值等同於你在該場冒險中從〈獻祭〉累計得到的掌握數量。舉行這種惡魔般的邪惡獻祭儀式極其危險，切勿輕率為之。
{{</ move-block >}}



## 進階能力
### 〈元素掌控〉{.move-title .withCheckbox}
{{< move-block >}} 
你可以控制元素，即使它們被用來對付你也一樣。這個動作視為戰鬥動作，並使用你的〈煉獄巫術〉加值：

若骰出 6 或以下，什麼事也沒發生。\
若骰出 7-9，使所有元素攻擊（火、冰、閃電等）對你造成的傷害減輕 1 點。\
若骰出 10-12，使這類攻擊對你造成的傷害減輕 2 點，或對你身邊同伴的傷害減輕 1 點。\
若骰出 13+，使這類攻擊對你造成的傷害減輕 3 點，或對你身邊同伴的傷害減輕 2 點。

如果你使這類攻擊的傷害減輕至 0 點，你可以將該攻擊改為轉向你選擇的一個新目標身上。
{{</ move-block >}} 

### 〈惡性折磨〉{.move-title .withCheckbox}
{{< move-block >}} 
你通曉該如何打擊對手以產生劇痛的方法。每當你對一個生物造成傷害時，該生物會因為受到劇痛折磨，而使你選擇的一個同伴在對付牠的任意動作中得到 單次+1。
{{</ move-block >}} 

### 〈即興供品〉{.move-title .withCheckbox}
{{< move-block >}} 
你擁有一把裝飾詭異的祭祀短刀（<etag>3 傷害</etag> <etag>近身</etag> <etag>鐵</etag>）。每當你使用這把短刀成功在戰鬥中殺死一個生物，或當一名受益於你所提供加值的同伴在戰鬥中殺死一個生物時，你都將立即得到能用於〈獻祭〉能力的 1 掌握，且這個掌握不會被計入你未來進行〈獻祭〉動作時的減值。
{{</ move-block >}} 

### 〈蛻變〉{.move-title .withCheckbox}
{{< move-block >}} 
你可以在短時間內化身成惡魔型態。消耗你〈獻祭〉動作的 1 掌握。在該場景期間，你得到利爪（<etag>4 傷害</etag> <etag>近身</etag> <etag>火焰</etag>）和硬皮（1 護甲，不會被非魔法的火焰和寒冰所傷害）。如果你額外消耗 1 掌握，則你還能得到在〈浴血奮戰〉動作的 持續+1，且變身型態將持續到下個日出或日落。
{{</ move-block >}} 

### 〈獵食罪惡〉{.move-title .withCheckbox}
{{< move-block >}} 
當你在〈訪談詢問〉動作中骰出至少 7 或以上的結果時，你可以免費詢問「這個人對什麼最有罪惡感？」。當你利用這股罪惡感來情緒勒索時，你的〈說服〉動作得到 持續+1。
{{</ move-block >}} 

### 〈獻祭專家〉{.move-title .withCheckbox}
{{< move-block >}} 
你的〈獻祭〉動作得到 持續+1。此外，你可以選擇消耗你〈獻祭〉上的掌握來讓任何動作得到 單次+1，但同一次擲骰中只能消耗一次（除了〈巫術〉動作，其仍能如常消耗複數掌握來得到加值）。
{{</ move-block >}} 

### 〈從餘燼中篩查〉{.move-title .withCheckbox}
{{< move-block >}} 
將以下問題加入你〈四處探查〉的問題列表中。
當你在〈四處探查〉動作的擲骰中得到了 7+ 的結果，你可以免費詢問其中一個：

- 這裡有什麼事物被摧毀了？
- 為什麼這裡的東西會被摧毀？
- 這起破壞發生在多久之前？
{{</ move-block >}} 

### 〈灰燼之魂〉{.move-title .withCheckbox}
{{< move-block >}} 
你出賣了你的靈魂，或為了得到你渴求的力量而將其燃燒殆盡。你在用於抵抗效果時進行的〈抵抗黑魔法〉動作得到 +2 加值，但再也不能斥退任何生物。你免疫於任何涉及吸收或竊取你靈魂的效果。
{{</ move-block >}} 

### 〈巫咒學家〉{.move-title .withCheckbox}
{{< move-block >}} 
選擇一種魔法：星界、死靈。你使用該種類魔法的〈巫術〉加值增加到 +2。
你知曉二種該學派魔法的戰鬥魔咒；且每場冒險一次，你可以施展一次儀式巫術魔咒而無須借助卷軸或其他外部資源的協助。
{{</ move-block >}}
