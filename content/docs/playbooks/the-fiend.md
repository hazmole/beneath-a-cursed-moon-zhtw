# 惡魔 Fiend

{{< playbook-desc >}}
並非所有惡魔都邪惡無比。出於某種原因，他們之中有些人決定改過向善。\
或許是因為他們在年紀尚幼時便被人類召喚並撫養長大；或許他們只是被邪魔附身的普通人；\
或許他們與人類墜入愛河；又或者他們打從一開始就對使壞做惡興致缺缺。

如今，他們以正義之名降下地獄烈火與詛咒，同時掙扎抵抗著體內的黑暗本性。
{{< /playbook-desc >}}


## 快速開局背景
{{% columns ratio="4:3" %}}
- <div class="withCheckbox subContentBlock">

  你的惡魔特徵是如此顯著：
  - 怪異的膚色。
  - 蹄足和尾巴。
  - 粗大犄角。

  你之所以走上偏離邪惡的道路，是因為：
  - 你從小就被人類扶養長大。
  - 你愛上了普通人。
  - 你親眼目睹了慘痛的悲劇。
  - 你從未有過邪惡的內心。

  其他 PC 與你相遇的契機是：
  - 他們懷疑你是應當被獵殺的怪物。
  - 你從其他惡魔手中拯救了他們。
  - 你與他們在同個組織中工作。

## 基本能力
### 〈違背天性〉{.move-title}
{{< move-block >}}
每當你透過一個戰鬥動作，在不造成傷害的情況下為了保護其他人而挺身而出，使自己深陷險境時，你下一個進行的動作將得到 單次+1。每當你進行〈說服〉動作，且在非必要的情況下對另一方表現出慷慨和誠實，治癒 1 點受到的傷害。
{{< /move-block >}}

### 〈惡魔表皮〉{.move-title}
{{< move-block >}}
你擁有 1 護甲（被帶有 <etag>魂縛</etag> 標籤的武器給無視）。你不會被非魔法的火焰和寒冰所傷害，且不會受到極端溫度的影響。
{{< /move-block >}}

### 〈地獄妖法〉{.move-title}
{{< move-block >}}
你知曉所有煉獄戰鬥巫術；且每場冒險一次，你可以施展一次儀式巫術魔咒而無須借助卷軸或其他外部資源的協助。
{{< /move-block >}}

### 〈煉獄武器〉{.move-title}
{{< move-block >}}
你擁有某種能噴發出烈焰的天生武器：利爪、重拳、犄角、棘刺等等。它們被視為具備以下屬性的武器：\
（<etag>4 傷害</etag> <etag>近身</etag> <etag>火焰</etag>）。
{{< /move-block >}}



## 進階能力
### 〈沉思〉{.move-title .withCheckbox}
{{< move-block >}} 
有時候，你就是需要一點空間獨處，憂悶地思考著你的黑暗身世 ── 但你總是能在需要時及時現身。當要執行調查行動時，你可以改為選擇進行〈沉思〉。這麼做能讓你在其中一個同伴們為他的調查動作擲骰後給他 單次+1，代表你一直在旁協助他。或者，如果有人在調查期間捲入麻煩，你也可以現身支援．並在你為了救他脫困的第一個行動中得到 單次+1。
{{</ move-block >}} 

### 〈邪氣魅力〉{.move-title .withCheckbox}
{{< move-block >}} 
儘管 ── 或正因為 ── 你的魔性外貌，你散發著某種獨特的誘惑力。你可以消耗來自〈訪談詢問〉的一次提問機會，來要求某人幫你一個忙，只要他們認為這件事不會讓他們捲入任何麻煩就行。當你進行〈說服〉動作時，你總是可以提出「之後再報答你」作為代價。儘管你可以賴帳，但直到你履行承諾前，你都不能再對該目標或知道這件事的人使用〈說服〉動作。
{{</ move-block >}} 

### 〈吞噬靈髓〉{.move-title .withCheckbox}
{{< move-block >}} 
你可以吞食墮落者的靈魂。每當你殺死一個活著的生物，或當在有人類死亡的戰鬥結束時，你得到 1 掌握。你可以當場立即消耗它，或保留到戰鬥結束後再使用。這個掌握能用來詢問任意調查動作中的一個該生物的知識所能回答的問題，或是治癒 1 點傷害。
{{</ move-block >}} 

### 〈忍辱前行〉{.move-title .withCheckbox}
{{< move-block >}} 
你學會將內心恥辱轉化為動力的方法。每當你在任何動作中骰出 6 或以下的結果，你得到適用於所有動作的 持續+1，持續直到你骰出 10+ 的結果。
{{</ move-block >}} 

### 〈狂怒〉{.move-title .withCheckbox}
{{< move-block >}} 
每當有生物對你造成傷害，或對一個同伴造成 2 點或以上的傷害時，你在你下一個用於對付牠的動作中得到 單次+1，但若該動作允許，則你必須選擇對其造成傷害的選項。你無法透過此能力對同個怪物獲得超過 單次+1 的加值。
{{</ move-block >}} 

### 〈恐懼的氣味〉{.move-title .withCheckbox}
{{< move-block >}} 
將以下問題加入你〈訪談詢問〉的問題列表中。\
當你在〈訪談詢問〉動作的擲骰中得到了 7+ 的結果，你可以免費詢問其中一個：

- 這裡的人們最恐懼的是什麼？
- 我該做什麼才能讓這裡的人們恐懼我？
- 該怎麼做才能平息這裡的恐懼？
{{</ move-block >}} 

### 〈皮粗肉糙〉{.move-title .withCheckbox}
{{< move-block >}} 
你的皮膚變得更加厚實強韌 ── 或許膚色也隨之變化。你額外得到 1 護甲（被帶有 <etag>魂縛</etag> 標籤的武器給無視），但你的惡魔本質在凡人眼中將再也無所遁形。
{{</ move-block >}} 

### 〈超自然靈氣〉{.move-title .withCheckbox}
{{< move-block >}} 
你散發出的不安靈氣能讓野獸、惡魔、和妖精類型的僕役不敢接近或攻擊你，除非你先主動攻擊。你也可以如同使用〈抵抗黑魔法〉動作一樣來斥退野獸，但改為使用你的〈煉獄巫術〉加值來擲骰。
{{</ move-block >}} 

### 〈翅翼〉{.move-title .withCheckbox}
{{< move-block >}} 
你長有巨大的翅翼，能讓你以和步行或奔跑一樣快的速度來飛行、滑翔、和浮空。
{{</ move-block >}}
