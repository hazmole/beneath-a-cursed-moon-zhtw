# 惡棍 Scoundrel

{{< playbook-desc >}}
那些敢於挺身對抗怪物的人們並不都是英雄，甚至連他們自己也不會這麼想。\
── 強盜、海賊、扒手、攔路搶匪，諸如此類，\
畢竟，黑暗中永遠潛伏著更危險的東西。

一次與真正怪物的遭遇，就足以改變任何人的優先事項，讓他們改為專注於奪回屬於自己的黑夜。
{{< /playbook-desc >}}


## 快速開局背景
{{% columns ratio="4:3" %}}
- <div class="withCheckbox subContentBlock">

  你粗獷強悍的外表與你的特徵是如此相襯：
  - 橫跨臉部的一道巨大傷痕。
  - 頭巾和敞開的襯衫。
  - 毫不掩飾的隨身刀具。

  你之所以拋棄罪犯生涯，是因為：
  - 某個你曾試圖行搶的人給了你第二次機會。
  - 某次搶案發生了慘痛的意外。
  - 怪物襲擊了你在乎的人們。

  你與另一位英雄的相遇，是因為：
  - 你當時試圖偷走他的東西。
  - 他聽說了你可怕的名號而前來尋求協助。
  - 你在錯誤的時間，錯誤的地點跟他在一起。
  - 你主動尋求他的指引，想學習如何回歸正道。
  - 他在你深陷困窘麻煩時拯救了你。

## 基本能力
### 〈伏擊高手〉{.move-title}
{{< move-block >}}
你在試圖埋伏或準備陷阱時的〈追獵〉動作得到 持續+1。
{{</ move-block >}}

### 〈出奇制勝〉{.move-title}
{{< move-block >}}
每當你攻擊一個未察覺到你的存在，或是尚未意識到你具有威脅性的目標時，你造成 傷害 +2。
{{</ move-block >}}

### 〈惡棍裝備〉{.move-title}
{{< move-block class="withCheckbox" >}}
你擁有一套輕型皮甲或鎖子甲，能在穿戴時提供 1 護甲。\
你還擁有一系列輕型武器，包括飛刀或投擲斧（<etag>3 傷害</etag> <etag>投擲</etag> <etag>鐵</etag>）以及以下其中一項：
- 一把使用鐵製彈藥的小型手弩或手槍（<etag>3 傷害</etag> <etag>遠程</etag> <etag>隱蔽</etag>）
- 數把匕首（<etag>3 傷害</etag> <etag>近身</etag> <etag>隱蔽</etag> <etag>鐵</etag>）
{{</ move-block >}}

### 〈滑頭的混蛋〉{.move-title}
{{< move-block >}}
每當你使用〈浴血奮戰〉動作，且沒有選擇造成傷害時，直到你的下一次行動之前，任何來源對你造成的傷害都會減輕 1 點。
{{</ move-block >}}



## 進階能力
### 〈人脈〉{.move-title .withCheckbox}
{{< move-block >}} 
每當你需要協助，且身處可能存在某些老同伴或其他賊團的環境時，你可以擲骰使用〈說服〉動作的加值：

若骰出 6 或以下，你得不到需要的協助，且消息會四處傳開。\
若骰出 7-9，你能弄到一些基本的補給品（對方不會多加追究），或一兩個願意幫你的傢伙。\
若骰出 10-12，你可以得到專業的（非法）協助，或幾個不介意弄髒雙手的武裝打手。\
若骰出 13+，你可以召集一整支幫派。
{{</ move-block >}} 

### 〈犯罪學問〉{.move-title .withCheckbox}
{{< move-block >}} 
當進行調查時，如果你在任何調查動作中骰出至少 7 或以上的結果，則你可以免費詢問以下其中一個問題，並能消耗來自任意調查動作的提問機會來詢問這些問題：

- 這裡最值錢的東西是什麼？
- 這裡是否曾發生過任何舞弊、騙局或陰謀？
- 這裡是否有人正在使用偽裝身分？
- 這個區域被動了什麼手腳？
{{</ move-block >}} 

### 〈揭露弱點〉{.move-title .withCheckbox}
{{< move-block >}} 
當你對敵人造成傷害，且擁有來自〈理解戰況〉的至少 1 掌握，則你無須消耗掌握就可以無視該敵人的 1 點護甲（最低降到 0）。其他同伴能夠隨時消耗來自〈理解戰況〉的 2 掌握來給你〈理解戰況〉的 1 掌握。
{{</ move-block >}} 

### 〈血流如注〉{.move-title .withCheckbox}
{{< move-block >}} 
每當你使用〈浴血奮戰〉來讓你的同伴得到 單次+1，你同時能在敵人身上造成一道流著血的傷口。除了原本的效果之外，你再造成 1 傷害（無視護甲），並在涉及該目標跡而進行的〈追獵〉和〈四處探查〉動作中，因為留下的血跡而得到 持續+1，且只要能看見傷口，你就能辨識出該目標的身分，即使牠試圖偽裝或變身也一樣。
{{</ move-block >}} 

### 〈惡名〉{.move-title .withCheckbox}
{{< move-block >}} 
你在道上流傳著某種名聲。你在利用這個名聲來進行〈說服〉動作時得到 持續+1，且能在進行〈訪談詢問〉動作時能得到一次額外的提問機會（即使你骰出了 6 或以下也一樣）；然而，這麼做可能會在對話結束後把目標嚇跑，或甚至引來當地執法機關的注意。
{{</ move-block >}} 

### 〈風險家〉{.move-title .withCheckbox}
{{< move-block >}} 
當你在〈大膽行動〉中骰出 10-12 的結果時，你可以藉由承擔一個後果，來額外獲得一個你在骰出 13+ 時可以選擇的好處。此外，你因為〈大膽行動〉的後果而承受的傷害上限永遠為 1 點。
{{</ move-block >}} 

### 〈惡棍的好運〉{.move-title .withCheckbox}
{{< move-block >}} 
每當你將要受到傷害，你都可以透過翻滾閃躲、或讓自己陷入窘境等形式，藉由承受下個動作的 單次-1，來減輕 1 點傷害。如果你使用這個能力來減輕傷害，則直到你重新站穩腳步並失去該減值為止，你都不能再次使用這個能力。
{{</ move-block >}} 

### 〈順手牽羊〉{.move-title .withCheckbox}
{{< move-block >}} 
每當你使用〈四處探查〉動作並得到至少 1 掌握，你都可以免費詢問「這裡藏著什麼東西？」或「這裡有什麼東西可能派得上用場？」，並在無人注意的情況下把它摸走。此外，如果你之後才發現這裡有什麼東西是你需要或可以偷走的，則你視為早就把它摸走了。
{{</ move-block >}} 

### 〈你到底站哪一邊？〉{.move-title .withCheckbox}
{{< move-block >}} 
你在抵抗會讓你傷害或背叛同伴的效果影響時而進行的〈抵抗黑魔法〉動作得到 +2 加值。然而，你相當擅長假裝屈服。每當你在試圖假裝背叛或喪失行動能力（無論是否涉及魔法）時，你在所有為了讓這件事看起來逼真可信的行動都將得到 持續+1；此外，你對相信你演技的對象進行的〈浴血奮戰〉得到 單次+1，並能對該目標進行伏擊。
{{</ move-block >}}
