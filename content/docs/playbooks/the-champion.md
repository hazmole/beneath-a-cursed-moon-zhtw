# 勇士 Champion

{{< playbook-desc >}}
他們秉持真理與正義，奉行自我犧牲的強大信念，對黑暗無所畏懼。\
他們拒絕狡詐與虛偽，挺胸邁步投入戰場以擊潰邪惡勢力，保護無辜的人們。\
而儘管勇氣能成為堅強之盾，正義能化作鋒利寶劍，披上沉重的全副武裝也同樣不可或缺。
{{< /playbook-desc >}}

## 基本能力
### 〈重型武裝〉{.move-title}
{{< move-block class="withCheckbox" >}}
你擁有一套絕讚的重型裝甲（可能還有一面盾牌），使你得到 2 護甲，以及下列其中一件武器：
- 戰斧或闊刃大劍（<etag>4 傷害</etag> <etag>近身</etag> <etag>鐵</etag>）
- 長矛（<etag>4 傷害</etag> <etag>及遠</etag> <etag>鐵</etag>）
- 短劍（<etag>3 傷害</etag> <etag>近身</etag> <etag>鐵</etag>）和 一把附有鐵製彈藥的長弓、弩或火槍（<etag>4 傷害</etag> <etag>遠程</etag>）
{{< /move-block >}}

### 〈保護弱小〉{.move-title}
{{< move-block >}}
每當你使用〈浴血奮戰〉動作來把注意力吸引到自己身上，你可以選擇讓一個你保護的同伴獲得對你交戰對手的 單次+1，或是使該生物下次對你造成的傷害減輕 1 點。
{{< /move-block >}}

### 〈驅趕邪惡〉{.move-title}
{{< move-block class="withCheckbox" >}}
每當你成功使用〈抵抗黑魔法〉來斥退邪惡生物，你可以選擇對該生物造成 1 傷害（無視護甲），或是得到對該生物的 單次+1。每場冒險一次，你可以製造一個你所選擇的驅魔工具：
- 火焰聖徽
- 鐵製聖徽
- 銀製聖徽
- 魂縛聖徽
{{< /move-block >}}



## 進階能力

### 〈權威〉{.move-title .withCheckbox}
{{< move-block >}} 
你持掌著被賦予的權威──這可能是源自某條廣泛認可的法律，或是僅在你所屬的組織內有效。\
在冒險開始時，骰 **+說服** 來進行此動作。

若骰出 7-9，得到 1 掌握。\
若骰出 10-12，得到 2 掌握。\
若骰出 13+，得到 3 掌握。

在該場冒險期間，你可以消耗 1 掌握來找到某個服從你權威的人物，並向他提出一個請求，視同你在〈說服〉動作中骰出了 10 的結果。
{{</ move-block >}}

### 〈注目焦點〉 {.move-title .withCheckbox}
{{< move-block >}} 
在進行〈追獵〉的擲骰前，你可以選擇把自己當作誘餌。如果你這麼做，則你的同伴在進行各自的〈追獵〉擲骰時將得到 單次+1，但你無法得到比 7-9 更好的結果。
{{</ move-block >}} 

### 〈信念指引〉 {.move-title .withCheckbox}
{{< move-block >}} 
當你以不涉及背叛、狡詐、欺瞞、或意外的形式進行〈大膽行動〉動作時，你總是能達成你設立的目標：骰出 6 或以下的結果時將視為骰出 7-9 。
{{</ move-block >}} 

### 〈過份正直〉 {.move-title .withCheckbox}
{{< move-block >}} 
只要你在這場冒險中還沒有對凡人說過謊，並且意圖信守承諾，則你在〈訪談詢問〉和〈說服〉動作中得到 +1 加值。
{{</ move-block >}} 

### 〈虛心求教〉 {.move-title .withCheckbox}
{{< move-block >}} 
你擅長聆聽，並願意尊重他們正確的意見。當其他人消耗〈理解戰況〉的掌握來特地幫助你時，他的下一次〈理解戰況〉動作將得到 單次+1（這個效果不會重複疊加）。此外，當你與其他人一起使用〈通曉傳說〉動作，且他在〈通曉傳說〉動作的加值比你高時，你在該動作中也能得到 單次+1。
{{</ move-block >}} 

### 〈領導力〉 {.move-title .withCheckbox}
{{< move-block >}} 
你可以指揮你的同伴完成任務，讓他們維持秩序、幹勁和熱忱。你可以不進行戰鬥或調查動作，而是改為選擇領導你的同伴，讓他們的下個動作得到 單次+1。
{{</ move-block >}} 

### 〈閃耀典範〉 {.move-title .withCheckbox}
{{< move-block >}} 
每當你在〈抵抗黑魔法〉動作中成功抵禦了危險的效果，或在〈大膽行動〉中骰出至少 7 的結果以躲避危險或達成壯舉，你的同伴在這個場景期間，都將在進行相同任務的動作中得到 持續+1。這個能力不會使他們在同個動作中得到超過 持續+1 的持續加值。
{{</ move-block >}} 

### 〈堅守屹立〉 {.move-title .withCheckbox}
{{< move-block >}} 
每當其他生物對你造成傷害，則你下一次對該生物造成 +1 傷害。
{{</ move-block >}} 

### 〈強大的靈魂〉 {.move-title .withCheckbox}
{{< move-block >}} 
所有你持用的武器都被視為擁有 魂縛 標籤，且你永遠不需要使用魂縛驅魔工具。此外，每當你讓隊友的戰鬥或巫術動作得到單次加值，則任何他們造成的傷害都將被視為使用了魂縛武器。
{{</ move-block >}}
