# 死靈法師 Necromancer

{{< playbook-desc >}}
操弄生與死的魔法是一條無比誘人的道路，應允人類能擊敗自己最大的敵人——死亡。

透過不死來追求永生的渴望，吸引了許多人投身死靈學派的黑暗魔法；\
而也有些人則致力於研究這種魔法來延續他人的性命，或試圖帶回逝去的摯愛。\
無論動機如何，死靈魔法都會緩緩啃食你的靈魂，讓施法者變得如同他們麾下的生物一般死氣沉沉。
{{< /playbook-desc >}}

## 基本能力
### 〈命令不死者〉{.move-title}
{{< move-block >}}
你可以嘗試統御那些不具心智的不死者們。這個動作視為戰鬥動作，並使用你的〈死靈巫術〉加值：

若骰出 6 或以下，你吸引了不死者的注意，使它們轉而向你襲來。\
若骰出 7-9，你可以讓不具心智的不死者僕役們暫時停止動作，跳過其下一個行動。\
若骰出 10-12，你可以讓不具心智的不死者衛士們暫時停止動作，或讓僕役們在下一個行動中為你而戰。\
若骰出 13+，駭人的不死者衛士將在下一個行動中為你而戰，且僕役們在整個場景期間都將聽從你的命令。

你可以持續對同一批不死者使用這個動作，直到你骰出 6 或以下的結果；一旦如此，它們將變得免疫於這個效果直到你好好休息。你每次使用這個能力時，都只能影響位於近身距離內的一小群不死者。
{{</ move-block >}}

### 〈為了知曉死亡〉{.move-title}
{{< move-block >}}
每當一個非僕役的生物在你附近死去，你都能得到用於〈理解戰況〉的 1 掌握，且只能用於對付與死去生物相同類型的生物（異怪、野獸、惡魔、妖精、不死者或人類）。
{{</ move-block >}}

### 〈死靈術者〉{.move-title}
{{< move-block >}}
你知曉所有死靈戰鬥巫術，且如果你想要施展死靈儀式巫術，你也清楚知道該儀式的內容，而無須再去尋找其相關紀錄或卷軸。
{{</ move-block >}}



## 進階能力
### 〈束縛靈魂〉{.move-title .withCheckbox}
{{< move-block >}}
當有生物在你附近死亡時，你可以選擇將他的靈魂囚禁在一個你親手製作的護符中。只要你帶著這個護符，你就能對這個靈魂展開審訊，使用〈訪談詢問〉或〈通曉傳說〉來得知它所知的一切。如果你花費整個晚上的時間舉行儀式，那麼這個靈魂將被永久囚禁；否則，它會在你下次使用〈束縛靈魂〉時被釋放。
{{</ move-block >}}

### 〈血祭〉{.move-title .withCheckbox}
{{< move-block >}}
使用你自己的鮮血作為儀式媒介，你能夠增強你的魔咒。如果所有和你一起參與儀式的成員都承受 1 點傷害（無視護甲），則你可以選擇忽略該儀式的其中一項要求。當你施放巫術來造成傷害時，你也可以選擇承受 1 點傷害來額外造成 傷害+2。
{{</ move-block >}}

### 〈詛咒大師〉{.move-title .withCheckbox}
{{< move-block >}}
每當你使用儀式巫術來對其他生物降下詛咒、惡咒或類似的效果時，你可以忽略該儀式的其中一項要求。每當你使用戰鬥巫術對其他生物造成任何非傷害性的負面影響時，該負面影響的持續時間會翻倍。
{{</ move-block >}}

### 〈黑暗轉化〉{.move-title .withCheckbox}
{{< move-block >}}
你已經成功蛻變成了不死者。你在〈抵抗黑魔法〉動作中得到 +2 加值，但再也不能斥退任何生物。除非是出於自衛或被其控制者明確下令，否則不具心智的不死者將不會主動攻擊你。你現在被視作不死者，且不再需要進食、飲水、睡眠或呼吸，但仍然可以如常治癒自己承受的傷害。
{{</ move-block >}}

### 〈死亡視界〉{.move-title .withCheckbox}
{{< move-block >}}
你可以看見生死之力的流轉。將以下問題加入你〈通曉傳說〉的問題列表中。
當你在〈通曉傳說〉動作的擲骰中得到了 7+ 的結果，你可以免費詢問其中一個：

- 這是誰或什麼東西的血液（或身體部位），它是否還活著？
- 這個人在何時死亡？又是怎麼死的？
- 這個人在生前經歷的最後一件事是什麼？
- 這個人或生物是不死者嗎？
{{</ move-block >}}

### 〈恐懼醫官〉{.move-title .withCheckbox}
{{< move-block >}}
每當你透過死靈巫術來治癒自己承受的傷害時，你可以改為治癒附近的同伴。如果你不打算改變你透過巫術治癒的目標（無論原本目標是自己或其他人），則可以再額外治癒 1 點傷害。
{{</ move-block >}}

### 〈精華聚焦〉{.move-title .withCheckbox}
{{< move-block >}}
如果你持有一個屬於某個生物的身體部位（包括毛髮、血液或其它），你就可以用它來引導你的魔法。只要該生物在你附近，你的魔咒就能以該生物為目標，即使你看不見牠也一樣。此外，你對持有其身體部位的該生物進行的〈追獵〉動作將得到 持續+1。
{{</ move-block >}}

### 〈收割者〉{.move-title .withCheckbox}
{{< move-block >}}
你在〈浴血奮戰〉動作中得到 持續+2，但如果你選擇使用這個加值，則你必須選擇造成傷害。\
你同時獲得一把重型武器：戰斧、闊刃大劍或鐮刀（<etag>4 傷害</etag> <etag>近身</etag> <etag>鐵</etag> <etag>魂縛</etag>）。
{{</ move-block >}}

### 〈巫咒學家〉{.move-title .withCheckbox}
{{< move-block >}}
選擇一種魔法：星界、煉獄。你使用該種類魔法的〈巫術〉加值增加到 +2。\
你知曉二種該學派魔法的戰鬥魔咒；且每場冒險一次，你可以施展一次儀式巫術魔咒而無須借助卷軸或其他外部資源的協助。
{{</ move-block >}}
