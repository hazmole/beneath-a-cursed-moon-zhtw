# 吸血鬼 Vampire

{{< playbook-desc >}}
承擔著不死詛咒，吸血鬼仰賴鮮血生存，並被迫獵食他人。\
有些人的意志堅強，足以抵抗這份詛咒\
── 或者得到了些許幫助 ── 因而保留了一部分的人性本質。

他們依然能感受到那股嗜血的呼喚，但仍奮力抗爭，竭盡所能地運用那份被賜予的黑暗力量。
{{< /playbook-desc >}}

## 基本能力
### 〈全副武裝〉{.move-title}
{{< move-block class="withCheckbox" >}}
你擁有一套輕型皮甲或鎖子甲（1 護甲），並擁有以下其中一項：
- 一把製作精良的決鬥用劍（<etag>4 傷害</etag> <etag>近身</etag> <etag>鐵</etag>）
- 一把長矛（<etag>4 傷害</etag> <etag>及遠</etag> <etag>鐵</etag>）
{{< /move-block >}}

### 〈鮮血點〉{.move-title}
{{< move-block >}}
吸血鬼的詛咒使你渴望活物的鮮血，這股衝動驅使著你，但也賦予你力量。\
你在每場冒險開始時擁有 3 點 鮮血點，並可以將其用於你的各種能力，最低消耗到 0 點。

你可以消耗 1 點鮮血點以在任意動作中得到 單次+1。你不能在被陽光直射的情況下消耗鮮血點，但可以在陰影或沒有太陽光的室內這麼做。如果某個動作指示你把鮮血點設為特定的數值，則你只有在當前鮮血點數量低於該數值時才需要這麼做。
{{</ move-block >}}

### 〈獠牙〉{.move-title}
{{< move-block >}}
你有一副鋒利的獠牙（<etag>3 傷害</etag> <etag>近身</etag>），可以用來吸取他人的血液。每當你使用獠牙造成傷害時，將你的鮮血點設為等同於該次傷害（在扣除護甲之後）的數值。如果你打算吸取自願或無力反抗的生物的鮮血，請改為使用〈共享鮮血〉動作。
{{</ move-block >}}

### 〈共享鮮血〉{.move-title}
{{< move-block >}}
你可以從自願或無力反抗的生物身上吸取鮮血。預先決定你想造成的傷害量（這個傷害無視護甲），並將你的鮮血點設為等同於該數量的數值。當你在鮮血點為 0 的時候使用〈共享鮮血〉，則你必須在宣告你想吸取的鮮血份量後，在無加值的情況下進行擲骰：

若骰出 6 或以下，你造成的傷害比預期還要多 2 點（並額外得到 2 點鮮血點）。\
若骰出 7-9，你造成的傷害比預期還要多 1 點（並額外得到 1 點鮮血點）。\
若骰出 10 或以上，你吸取的血液如同預期。
{{</ move-block >}}

### 〈褻瀆妖法〉{.move-title}
{{< move-block >}}
你知曉所有死靈戰鬥巫術；且每場冒險一次，你可以施展一次儀式巫術魔咒而無須借助卷軸或其他外部資源的協助。
{{</ move-block >}}


## 進階能力
### 〈野獸形態〉{.move-title .withCheckbox}
{{< move-block >}} 
藉由消耗 1 點鮮血點，你可變形成一頭狼或蝙蝠。當變形成狼時，你在〈四處探查〉和〈追獵〉動作中得到 持續+1，並保有你的〈獠牙〉。當變形成蝙蝠時，你能以和奔跑相同的速度飛行，但無法進行任何攻擊。你在變形期間無法說話、穿戴護甲、或使用大部分的道具和武器，但任何你原本攜帶的物品都將融入新形態中。變回人類形態並不需要消耗任何鮮血點。
{{</ move-block >}} 

### 〈生命之血〉{.move-title .withCheckbox}
{{< move-block >}} 
你能利用你的不死詛咒來治癒傷口。你可以消耗 1 點鮮血點來治癒 1 傷害。以一個戰鬥動作，你也能讓其他人飲用你的鮮血，藉由每讓自己承受 1 傷害來使對方治癒 1 傷害。
{{</ move-block >}} 

### 〈夜之斗篷〉{.move-title .withCheckbox}
{{< move-block >}} 
黑暗環繞並守護著你。每當你與陽光徹底隔絕時（像是在夜晚、地底深處或沒有窗戶的房間），你得到會被 魂縛 武器給無視的 1 護甲。此外，你在即使最深邃的黑暗中也能清晰視物。
{{</ move-block >}} 

### 〈暗黑魅力〉{.move-title .withCheckbox}
{{< move-block >}} 
你身懷著超自然的吸引力。每當你使用〈訪談詢問〉動作來從某人身上問出情報時，你對該目標進行的〈說服〉動作也將得到 持續+1。你可以使用〈說服〉動作來誘使人們讓你飲用他的鮮血，且只要你沒有殺死他，他將從中感受到足以讓他願意再度獻身的快感。
{{</ move-block >}} 

### 〈嶄露獠牙〉{.move-title .withCheckbox}
{{< move-block >}} 
每當你要攻擊一個未察覺到你存在的敵人，或你在該〈浴血奮鬥〉動作中得到了來自其他隊友的單次加值時，你的〈獠牙〉提升到 4 傷害。
{{</ move-block >}} 

### 〈迷霧形態〉{.move-title .withCheckbox}
{{< move-block >}} 
藉由消耗 1 點鮮血點，你可以變形成一團雲霧，讓你能夠在空中穿梭並鑽入微小隙縫。當處於雲霧形態時，你能以和步行相同的速度飛行，無法進行任何形式的攻擊，且不會受到除了火焰以外的任何傷害。
{{</ move-block >}} 

### 〈狩獵鮮血〉{.move-title .withCheckbox}
{{< move-block >}} 
你學會了透過狩獵野生動物來平息你的飢渴。以一個調查動作，使用〈追獵〉的加值進行擲骰，若在夜晚則這個動作獲得 持續+1：

若骰出 6 或以下，你沒能找到任何可供吸食的獵物。\
若骰出 7-9，你找到了小型野生動物；將你的鮮血點設為 2 點。\
若骰出 10-12，你成功獵到了一頭鹿或類似尺寸的動物；將你的鮮血點設為 3 點。\
若骰出 13+，你獵到了能讓你大快朵頤的大型獵物；將你的鮮血點設為 4 點。
{{</ move-block >}} 

### 〈催眠凝視〉{.move-title .withCheckbox}
{{< move-block >}} 
你可以催眠直視你雙眼的生物，使其成為能輕易得手的獵物。使用〈說服〉的加值進行擲骰：

若骰出 6 或以下，你催眠的意圖太過明顯，被目標識破。\
若骰出 7-9，你可以催眠一個人類或野獸類型的僕役，持續直到你採取其他行動。\
若骰出 10-12，你可以在停止凝視後讓一個人類或野獸類型的僕役持續恍神長達數分鐘的時間，或讓一個衛士被催眠直到你採取其他行動。\
若骰出 13+，你誘使一個人類或野獸類型的僕役陷入恍惚狀態長達數小時的時間，或讓一個衛士恍神長達數分鐘。

你在對被你催眠的生物進行的〈浴血奮戰〉和〈說服〉動作中得到 單次+1。但被催眠的生物只要受到任何傷害就會中斷催眠的效果。
{{</ move-block >}} 

### 〈血腥怪力〉{.move-title .withCheckbox}
{{< move-block >}} 
每當你在擁有 2 點或以上鮮血點時消耗了鮮血點來得到 單次+1，你改為得到該動作（且僅限該動作）的 持續+1，持續直到該場景結束或你的鮮血點歸零。
{{</ move-block >}}
