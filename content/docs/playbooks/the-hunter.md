# 獵人 Hunter

{{< playbook-desc >}}
許多人以專業怪獸獵人的身份四處流浪。他們精通各種用來追蹤與殺死各類生物的技巧。\
很有可能是因為他們過去曾遭遇某種慘痛的變故，才讓他們決心投入這條道路，\
或許也是因此，總會有年長的獵人願意指導下一代，將這份技藝傳承下去。

或許，這份天職早已流淌在他們的血液之中。
{{< /playbook-desc >}}


## 快速開局背景
{{% columns ratio="4:3" %}}
- <div class="withCheckbox subContentBlock">

  你的獵人裝束是：
  - 沾有泥濘的長披風。
  - 因旅途勞頓而幾經磨損的貴族華服。
  - 胸甲和護腿裙甲。

  你之所以把人生投入怪物狩獵，是因為：
  - 你必須向殺害你家人的生物復仇。
  - 你的家族血脈世代都是如此。
  - 這不過是為了混口飯吃。

  你加入其他獵人的行列，是因為：
  - 你們曾跟隨同一位導師。
  - 你曾向其中一人傳授關於狩獵的技藝。
  - 你曾從怪物的襲擊中救下其中一人。
  - 你們的家族一直都是合作夥伴。
  - 你們曾為了狩獵同一隻怪物而聯手。
  - 你曾誤以為其中一人是你必須獵殺的怪物。

## 基本能力
### 〈輕裝甲〉{.move-title}
{{< move-block >}}
你擁有一套輕型皮甲或鎖子甲，能在穿戴時提供 1 護甲。
{{</ move-block >}}

### 〈窮追不捨〉{.move-title}
{{< move-block >}}
在使用〈四處探查〉動作來詢問關於怪物的問題之後，你下次對該怪物進行的〈追獵〉動作得到 單次+1。你無法透過此能力對同個怪物獲得超過 單次+1 的加值。
{{</ move-block >}}

### 〈招牌武器〉{.move-title}
{{< move-block class="withCheckbox" >}}
你起始擁有一把魔法武器，由你決定這把武器的具體細節。

這把武器或其彈藥的的材質為：
- <etag>鐵</etag>
- <etag>銀</etag>

且這把武器帶有其中一個標籤：
- <etag>火焰</etag>
- <etag>魂縛</etag>

從以下選擇一項：
- 戰斧或闊刃大劍（<etag>4 傷害</etag> <etag>近身</etag>）
- 鎖鍊鞭或長矛（<etag>4 傷害</etag> <etag>及遠</etag>）
- 弩、長弓或火槍（<etag>4 傷害</etag> <etag>遠程</etag>）
{{</ move-block >}}

### 〈滅殺者〉{.move-title}
{{< move-block >}}
你對其中一種類型的怪物瞭若指掌──也許牠們是你受訓時的主要假想敵、也許是因為牠們殺害了你的家人，焚毀你的家園，或也許是你的血脈中流淌著一縷迫使你與之對抗的魔法能量。

選擇一種怪物類型：異怪、野獸、惡魔、妖精、不死者。

你在對這類生物使用〈理解戰況〉、〈通曉傳說〉和〈抵抗黑魔法〉動作時得到 持續+1；此外，當你在調查這類生物時使用〈通曉傳說〉，你還能免費詢問「牠是什麼類型的生物？」而無須消耗掌握。最後，從〈行家工具〉中選擇一個與該怪物類型有關的道具 ── 你總是將其隨身備著。
{{</ move-block >}}


### 〈行家工具〉{.move-title}
{{< move-block >}}
你擁有一個裝滿可靠道具的行囊。每場冒險二次，你可以拿出以下列表中的任何道具。其中任何以複數形式呈現的道具（像是箭矢或飛刀）都足以供單人隨時取用，並維持長期消耗：
- 數把銀製飛刀（<etag>3 傷害</etag> <etag>投擲</etag> <etag>銀</etag>）
- 數把投擲斧（<etag>3 傷害</etag> <etag>投擲</etag> <etag>鐵</etag>）
- 一把隱匿匕首（<etag>3 傷害</etag> <etag>近身</etag> <etag>隱蔽</etag> <etag>鐵</etag>）
- 一把傳家寶劍（<etag>3 傷害</etag> <etag>近身</etag> <etag>魂縛</etag>）
- 一把重型火炬（<etag>3 傷害</etag> <etag>近身</etag> <etag>火焰</etag>）
- 數支銀製或鐵製彈藥
- 一件火焰聖徽（驅魔工具）
- 一件鐵製聖徽（驅魔工具）
- 一件銀製聖徽（驅魔工具）
- 一件魂縛聖徽（驅魔工具）
{{</ move-block >}}



## 進階能力
### 〈備用武器〉{.move-title .withCheckbox}
{{< move-block >}} 
選擇並獲得第二把〈招牌武器〉。\
你同時擁有這兩把武器，或者你只要手腕一甩，就能讓招牌武器在兩者之間變形切換。
{{</ move-block >}} 

### 〈更大的行囊〉{.move-title .withCheckbox}
{{< move-block >}} 
你每天可以從〈行家工具〉中額外拿出兩件道具。此外，把以下道具加入列表中：

- 數把塗毒小刀（<etag>2 傷害</etag> <etag>無視護甲</etag> <etag>投擲</etag>，僅對活物有效）
- 數瓶燃燒瓶（<etag>2 傷害</etag> <etag>遠程</etag> <etag>火焰</etag>，目標在撲滅火焰前每次行動都會受到無視護甲的 1 傷害）
- 一小瓶純淨靈魂精華（讓一把武器在一個場景期間得到 <etag>魂縛</etag> 標籤）
- 一小瓶燃油（讓一把武器在一個場景期間得到 <etag>火焰</etag> 標籤）
{{</ move-block >}} 

### 〈創造機會〉{.move-title .withCheckbox}
{{< move-block >}} 
每當你使用〈抵抗黑魔法〉動作成功斥退任意生物，或成功抵禦了該生物的力量效果影響時，你在你下一次對該生物進行的〈浴血奮戰〉動作中得到 單次+1。
{{</ move-block >}} 

### 〈血脈中的魔法〉{.move-title .withCheckbox}
{{< move-block >}} 
選擇一種與你〈滅殺者〉所選怪物類型有關的巫術種類：\
星界（異怪）、煉獄（惡魔）、死靈（不死生物），若所選怪物類型為妖精，則選擇任意一種。

你使用該種類魔法的〈巫術〉加值增加到 +2，且你可以從你的〈行家工具〉中拿出該學派魔法的戰鬥巫術卷軸或儀式巫術卷軸。
{{</ move-block >}} 

### 〈導師〉{.move-title .withCheckbox}
{{< move-block >}} 
每當你從〈行家工具〉中拿出一件道具並轉交給其他人時，你可以再拿出另一份同樣的道具 ── 看是要自己留著，還是再轉交出去。此外，每當你因為任何原因而得到 單次+1 時，你可以改為將這個加值轉交給另一個角色，該加值的限制和用途仍然保持不變。
{{</ move-block >}} 

### 〈聲名顯赫〉{.move-title .withCheckbox}
{{< move-block >}} 
身為怪物獵人，你經驗豐富且名聲遠播 ── 現在，當你試圖讓凡人協助你對付你在〈滅殺者〉所選的怪物時，〈滅殺者〉的加值也將同樣適用於〈說服〉或〈訪談詢問〉動作；同時，該加值也適用於對抗該類怪物的〈四處探查〉和〈追獵〉動作。
{{</ move-block >}} 

### 〈眾生滅殺者〉{.move-title .withCheckbox}
{{< move-block >}} 
為你的〈滅殺者〉額外選擇一種怪物類型。
{{</ move-block >}} 

### 〈邪惡的臭味〉{.move-title .withCheckbox}
{{< move-block >}} 
每當你使用〈訪談詢問〉動作時，你可以消耗一個問題的額度來詢問「我正在談話的對象是怪物嗎？」。如果牠屬於你〈滅殺者〉所選的怪物類型，則你會被告知其具體種類。
{{</ move-block >}} 

### 〈無盡追獵〉{.move-title .withCheckbox}
{{< move-block >}} 
你在為了追蹤、逼近、或任何讓你能追上目標以展開戰鬥的〈大膽行動〉動作中得到 單次+1，且你永遠可以選擇承受 1 點傷害（無視護甲）來做為該動作的後果。此外，當你是為了追上你〈滅殺者〉所選的敵人時，〈滅殺者〉的加值也將同樣適用於〈大膽行動〉。
{{</ move-block >}}
