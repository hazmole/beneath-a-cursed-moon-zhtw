# 調換兒 Changeling

{{< playbook-desc >}}
有些人會在誤入禁地的數年後歸來，身體毫無變化，卻永遠染上了妖精的氣息。\
又有時，孩童會從搖籃中被擄走，留下妖精之子取而代之。\
調換兒正是以這種方式連接著人類與妖精的世界\
——他們可能是踏入妖精之地的凡人，也可能是對人類世界產生了銘印的妖精。

他們發現儘管自己持擁著妖精的迷魅，卻因身處兩個世界的夾縫，總是被身邊的人們給遺忘。
{{< /playbook-desc >}}



## 快速開局背景
{{% columns ratio="4:3" %}}
- <div class="withCheckbox subContentBlock">

  你的妖精遺徵是如此明顯：
  - 墨黑色的雙眼。
  - 尖尖的長耳。
  - 超凡美貌。
  - 不自然的髮色。
  - 頭髮和肌膚會隨著季節變換。

  你成為了浪子：
  - 是在逃離妖精國度之後。
  - 是在你的雙親因為你的妖精遺徵而拒絕你之後。
  - 是因為浪跡四方的衝動如此強烈。

  你之所以加入其他英雄的行列：
  - 因為他們似乎是唯一能把注意力放在你身上的人們。
  - 因為當所有人背棄你時，他們之中有人接納了你。
  - 因為怪胎必須互相扶持。
  - 因為你發現自己深受其中一人吸引，並渴望待在他們身邊。

## 基本能力
### 〈被遺忘的孩子〉{.move-title}
{{< move-block >}}
每當你使用調查動作，你都可以讓任何人無法記住你的存在 ── 但他們在當下仍然會正常與你互動。如果你再花費來自任何調查動作的 1 掌握，則你可以讓你的同伴們也被一起遺忘。你可以自由決定要誰被記住，誰被遺忘。
{{< /move-block >}}

### 〈獵人工具〉{.move-title}
{{< move-block class="withCheckbox" >}}
你擁有以下其中一套武器，以及一件具有 1 護甲的輕型裝甲。
- 附有銀製箭矢的長弓或弩（<etag>4 傷害</etag> <etag>遠程</etag>）；一把短劍或伐木斧（<etag>3 傷害</etag> <etag>近身</etag> <etag>鐵</etag>）
- 銀製長矛（<etag>4 傷害</etag> <etag>及遠</etag> <etag>銀</etag>）；數把飛刀或投擲斧（<etag>3 傷害</etag> <etag>投擲</etag> <etag>鐵</etag>）
{{< /move-block >}}

### 〈無聲無息〉{.move-title}
{{< move-block >}}
你的存在很難引起其他生物的注意。除非你先出手攻擊或奪走他們的東西，否則野獸、惡魔和人類永遠不會攻擊你。當你與這類生物戰鬥時，你對他進行的第一個戰鬥動作將得到 單次+1，就如同你成功伏擊他一樣。
{{< /move-block >}}

### 〈低語〉{.move-title}
{{< move-block >}}
當你使用〈說服〉動作時，你可以讓自己在該擲骰中得到 單次-1，藉此讓目標遺忘他與你之間的互動，讓他誤以為這是出自他自己的想法。
{{< /move-block >}}


## 進階能力

### 〈迷魅之音〉{.move-title .withCheckbox}
{{< move-block >}} 
你的聲音能流露出迷惑人的性質。骰 **+說服** 來進行此動作。

若骰出 6 或以下，你使用魔法的舉動太過明顯，反過來激怒了你的目標。\
若骰出 7-9，只要你說話，就能讓人類和野獸類型的僕役恍神，但他們會在效果結束後意識到自己遭到耍弄。\
若骰出 10-12，只要你說話，就能讓人類和野獸類型的僕役或衛士恍神，並在數分鐘後仍不會察覺到異樣。\
若骰出 13+，則你的聲音能夠誘使人類和野獸類型的僕役或衛士陷入恍惚狀態長達數小時的時間，或直到有人將他們喚醒。
{{</ move-block >}} 

### 〈妖精皮膚〉{.move-title .withCheckbox}
{{< move-block >}} 
你的表皮強韌無比，能夠抵禦野獸的尖牙利爪。你得到 1 護甲（被帶有 <etag>鐵</etag> 標籤的傷害給無視）。
{{</ move-block >}} 

### 〈精族親緣〉{.move-title .withCheckbox}
{{< move-block >}} 
你在為了抵抗妖精生物的魔法影響時而進行的〈抵抗黑魔法〉得到 +2 加值（但不適用於斥退妖精生物）。此外，你在任何自然環境中進行的〈追獵〉動作也會得到 +1 加值。
{{</ move-block >}} 

### 〈失物發現者〉{.move-title .withCheckbox}
{{< move-block >}} 
你不會受到任何阻止你察覺人事物的魔法和幻術的影響。此外，當你進行〈四處探查〉動作（針對特定區域）和〈訪談詢問〉動作（向某個人物或群體打聽）時，將以下問題加入提問選項中：

- 這裡有什麼遺失了？──無論是實體或象徵意義上的。
- 人們有什麼尚未實現的心願？
- 人們對於在這裡逝去的什麼感到遺憾？
{{</ move-block >}} 

### 〈看不見的朋友〉{.move-title .withCheckbox}
{{< move-block >}} 
每當你使用戰鬥動作以把某個生物的注意力轉為吸引到自己身上時，你要保護的對象將被幻象遮蔽 ── 只要你仍在附近，且他不主動攻擊該生物，那他就會被你正在對付的生物徹底遺忘。被這個方式隱藏起來的同伴在對該生物進行的下一個動作將得到 單次+1，但在該場戰鬥中無法再次受益於〈看不見的朋友〉的效果。
{{</ move-block >}} 

### 〈聆聽荒野之聲〉{.move-title .withCheckbox}
{{< move-block >}} 
你可以和動物、植物及其他自然生物交談，並理解牠們的意思。當你使用〈訪談詢問〉動作時，你可以選擇向牠們打聽情報；你也可以使用〈說服〉動作來讓野生或被馴化的動物為你做事。
{{</ move-block >}} 

### 〈守誓者〉{.move-title .withCheckbox}
{{< move-block >}} 
你可以立下一個誓言，藉此獲得強大的能力來履行它。如果你立誓要保護某人或某物，則你在進行任何用於直接保護它不受傷害的動作中都能得到 單次+1，並能在它陷入危險時透過魔法得知這件事。如果你立誓要殺死某個存在，則你對該生物造成 +1 傷害。若你背棄或沒能履行你的誓言，則你在〈抵抗黑魔法〉、〈說服〉和〈巫術〉動作中承受 持續-1 的罰值，持續直到該場冒險結束。你在每場冒險中只能立下一個誓言。
{{</ move-block >}} 

### 〈王庭律法〉{.move-title .withCheckbox}
{{< move-block >}} 
無論知情與否，你都曾向妖精王庭的律法發過誓。當你以說實話、或不說謊的前提下使用誤導性的文字遊戲和詐術來進行〈說服〉動作時，你得到 持續+1 。只要你不說謊，你就能分辨他人是否在對你說謊 ── 如果你最終還是說了謊話，那麼你將失去這個能力的好處，並在〈說服〉動作中承受 持續-1 的罰值，持續直到這場冒險結束。
{{</ move-block >}} 

### 〈魔法祝福〉{.move-title .withCheckbox}
{{< move-block >}} 
選擇一種魔法：星界、煉獄、死靈。你使用該種類魔法的〈巫術〉加值增加到 +2。
你知曉二種該學派魔法的戰鬥魔咒；且每場冒險一次，你可以施展一次儀式巫術魔咒而無須借助卷軸或其他外部資源的協助。
{{</ move-block >}}
