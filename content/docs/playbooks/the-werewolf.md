# 狼人 Werewolf

{{< playbook-desc >}}
作為黑夜中最恐怖的野獸之一，狼人是獸性力量與憤怒的化身。\
那些受到狼化症感染的可憐人註定要永遠成為威脅，任憑體內的殺戮狂熱隨時爆發，撕碎一切。

其中意志最為堅定的人們學會了反抗，並最終奪回少許身體的控制權。\
即使對周遭的人們仍然危險萬分，但他們致力於確保那些被狼人怒火波及的對象，\
至少是足以應付這份力量的英雄 ── 或者是罪有應得的怪物。
{{< /playbook-desc >}}

## 基本能力
### 〈獸化形態〉{.move-title}
{{< move-block >}}
當你變形成你的獸化形態 ── 無論是大型狼或半人半獸的混合體 ── 你得到在〈浴血奮戰〉、〈追獵〉動作、以及著重於速度或蠻力等體能表現的〈大膽行動〉動作的 持續+1。你同時得到利爪和咬噬（<etag>5 傷害</etag> <etag>近身</etag>）以及 2 護甲（被帶有 <etag>銀</etag> 標籤的傷害給無視）。當處於獸化形態時，你無法進行〈理解戰況〉、〈通曉傳說〉、〈訪談詢問〉、〈說服〉或〈巫術〉動作。
{{</ move-block >}}

### 〈自我控制〉{.move-title}
{{< move-block >}}
你得到一個特殊動作〈自我控制〉，用來克制自己不轉變為獸化形態，也用來強迫自己變回原形。你起始的〈自我控制〉加值為 +2，但在滿月下會承受 持續-1，且在擁有 5 點或以上的傷害時再承受 持續-1。每當你受到傷害，或你在每個場景中第一次暴露在滿月的照射下時，你就必須進行〈自我控制〉動作：

若骰出 6 或以下，你變形成獸化形態。\
若骰出 7-9，你暫時壓制住了獸性。\
若骰出 10+，你在這個場景間都不需要再進行〈自我控制〉動作。

你可以隨時自願變形成你的獸化形態，且在〈自我控制〉動作失敗時也會被迫變形。一旦處理掉眼前的威脅，你就可以嘗試變回原形。如果你處於陽光照射下，那麼你將自動變回原形。如果是在滿月期間，則你需要等到天亮才能變回來。至於在其他狀況下，你可以在每場戰鬥結束時、或在每次調查動作之後進行一次〈自我控制〉動作，並在骰出 7+ 的結果時變回來。
{{</ move-block >}}

### 〈敏銳嗅覺〉{.move-title}
{{< move-block >}}
每當你進行〈四處探查〉動作時，你能得到額外的 1 掌握，且必須消耗其中一個問題額度來辨識生物的動向、數量或身分，或是用來詢問以下被加入〈四處探查〉問題列表的問題：

- 我在這裡聞到了誰的氣味？
- 我以前在哪裡聞過這個人的氣味？
{{</ move-block >}}

### 〈搜刮武器〉{.move-title}
{{< move-block >}}
你學會了如何用手邊的物資來自衛。\
只要花點時間，你就能弄來一把伐木斧（<etag>3 傷害</etag> <etag>近身</etag> <etag>鐵</etag>）或草叉（<etag>3 傷害</etag> <etag>及遠</etag> <etag>鐵</etag>）。
{{</ move-block >}}



## 進階能力
### 〈管理怒火〉{.move-title .withCheckbox}
{{< move-block >}} 
當你因為〈大膽行動〉的後果而受到傷害時，你不再需要為此進行〈自我控制〉動作。此外，你即使在正常人類形態下，也可以引導你的獸性怪力來達成驚人壯舉：你在著重於速度或蠻力等體能表現的〈大膽行動〉動作中得到 持續+1。這個加值會與〈獸化形態〉的加值疊加。
{{</ move-block >}} 

### 〈動物吸引力〉{.move-title .withCheckbox}
{{< move-block >}} 
你能夠理解動物們的需求和渴望，讓你能夠使用〈說服〉動作來讓野生或被馴化的動物為你做事。當面對狼或狗的時候，你得到〈說服〉動作的 持續+1。此外，即使你處於獸化形態，你也能夠對狼和狗進行〈說服〉和〈訪談詢問〉動作。
{{</ move-block >}} 

### 〈內在之獸〉{.move-title .withCheckbox}
{{< move-block >}} 
即使不處於獸化形態，你也能顯露出利爪（<etag>4 傷害</etag> <etag>近身</etag>）以及對傷害的抗性（1 護甲，被帶有 <etag>銀</etag> 標籤的傷害給無視），你的敏銳感官也將讓你在〈追獵〉動作中得到 +1 加值。當你變形時，這些效果會被獸化形態的加值給取代。
{{</ move-block >}} 

### 〈掙脫鎖鏈〉{.move-title .withCheckbox}
{{< move-block >}} 
當你處於獸化形態時，你能夠無視任何敵人試圖操控你心智的魔法效果影響。每當有人嘗試用魔法影響你，你都可以選擇立即變身成獸化形態。此外，每當你變身或恢復原形時，你都能治癒 1 傷害。
{{</ move-block >}} 

### 〈銀月之子〉{.move-title .withCheckbox}
{{< move-block >}} 
你的利爪和咬噬現在被視為帶有 銀 標籤的武器，且當處於獸化形態時，你的護甲對帶有 銀 標籤的攻擊而言只會視為比原本降低 1 點，而非被完全無視。
{{</ move-block >}} 

### 〈暴走〉{.move-title .withCheckbox}
{{< move-block >}} 
每當你使用你的尖牙或利爪攻擊時，你可以選擇讓〈浴血奮戰〉動作的效果波及到所有觸及範圍內的生物，視同其具備 群體 標籤，但你可以選擇讓一個對項不被波及 ── 這個效果同時適用於敵方和同伴。
{{</ move-block >}} 

### 〈撕裂〉{.move-title .withCheckbox}
{{< move-block >}} 
每當你使用你的尖牙或利爪對擁有護甲的生物造成傷害時，該生物的護甲在面對任何攻擊時都將被視為比原本減少 1 點，持續直到該場景結束。這個能力無法將生物的護甲減少超過 1 點。
{{</ move-block >}} 

### 〈月華術者〉{.move-title .withCheckbox}
{{< move-block >}} 
與你一樣被縛於銀月的魂體在對你低語。你的〈星界巫術〉加值增加到 +1，且只要月亮清晰可見，你的〈星界巫術〉就會再得到 +1 加值。你可以在處於獸化形態時使用〈星界巫術〉。
你知曉二種該學派魔法的戰鬥魔咒；且每場冒險一次，你可以施展一次儀式巫術魔咒而無須借助卷軸或其他外部資源的協助。
{{</ move-block >}}
